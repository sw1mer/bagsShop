Опции

Список, включающий все варианты настраиваемых опций

count

Тип: Number
Default: 10

До какого числа происходит умножение

title

Type: String
Default: 'ТАБЛИЦА'

Заголовок таблицы

captionAlign

Type: String
Default: 'text-center'
Допустимые значения: 'text-left'/'text-center'/'text-right'

Выравнивание заголовка таблицы

captionColor

Type: String
Default: 'black'

Цвет заголовка таблицы

captionSize

Type: String
Default: '1rem'

Размер шрифта заголовка
