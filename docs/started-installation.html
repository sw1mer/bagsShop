Установка

Подключение CSS

Сначала включите файл CSS в HTML head:

<link rel="stylesheet" href="multiply.css">

Подключение JS

Подключите скрипты в указанном порядке перед закрывающим тегом body

<script src="lodash.js"></script>
<script src="multiply.js">&n</script>

Структура HTML

Вам не нужна специальная разметка. Все, что вам нужно, это добавить контейнер. Вы можете использовать другой класс или id. В таком случае при подключении плагина необходимо указать соответствующий класс. 

<!-- Set up your HTML -->
<div class="multiply"></div>

Вызов плагина

Теперь вызовите функцию инициализации и ваша таблица умножения готова.

var multiply2 = new Multiply({ 
    elem: document.querySelector('.table-multiply'),
});
